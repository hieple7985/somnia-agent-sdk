# SomniaAgent SDK Landing Page

Professional landing page showcasing the AI Agent Development Framework for Somnia Blockchain.

## Quick Start

### Local Development

```bash
cd landing
python3 -m http.server 8000
```

Open http://localhost:8000

### Deploy to Vercel

```bash
cd landing
vercel --prod
```

## Structure

```
landing/
├── index.html
├── styles.css
├── script.js
└── README.md
```

- Hero section with key metrics
- Code comparison demo
